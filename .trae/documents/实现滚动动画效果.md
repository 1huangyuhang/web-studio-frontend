## 实现滚动动画效果

### 1. 组件设计
创建一个新的 `ScrollAnimation` 组件，支持两种动画效果：

**效果A：卡片入场动画**
- 初始状态：`translateY(30px) + opacity(0)`
- 入场动画：平滑过渡到 `translateY(0) + opacity(1)`
- 过渡时长：0.6s，缓动函数 `ease-out`
- 触发条件：元素进入视口（可见度≥50%），仅执行1次

**效果B：数字递增动画**
- 从 0 逐步递增到目标数字
- 带千分位格式化
- 动画时长：1.2s
- 触发条件：同效果A

### 2. 技术实现
- 使用 `IntersectionObserver API` 监听元素视口状态
- 为 `.animate-card` 类添加入场动画
- 为 `.animate-number` 类添加数字递增动画，通过 `data-target` 属性设置目标值
- 使用 `requestAnimationFrame` 实现平滑的数字递增
- 添加 CSS 变量控制动画参数

### 3. 文件修改
1. **创建新文件**：`/src/animations/ScrollAnimation.tsx` - 滚动动画组件
2. **创建新文件**：`/src/animations/ScrollAnimation.less` - 动画样式
3. **修改**：`/src/pages/Home/index.tsx` - 在统计数据区应用数字动画
4. **修改**：在需要卡片动画的地方添加 `.animate-card` 类

### 4. 核心逻辑
- 使用 `useIntersectionObserver` 自定义 hook 检测元素可见性
- 数字动画使用 `requestAnimationFrame` 实现平滑递增
- 卡片动画通过 CSS 过渡实现
- 支持千分位格式化
- 仅执行一次动画

### 5. 应用场景
- **卡片动画**：生态环境插件列表、应用案例板块
- **数字动画**：社区数据统计区（GitHub Stars、用户数等）
